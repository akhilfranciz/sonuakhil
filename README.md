# Simple Video Chat Application

A room-based video chat application with mobile responsive design.

## Features
- Room-based video calling
- Enter room number to join
- Multiple participants per room
- Mobile responsive design
- Video and audio controls
- Real-time WebRTC communication

## Installation

```bash
npm install
```

## Run

```bash
npm start
```

Then open http://localhost:3000 in your browser.

## Usage

1. Enter a room number on the home page
2. Click "Join Room"
3. Allow camera and microphone access
4. Share the room number with others to join
5. Use controls to toggle video/audio
6. Click "Leave Room" to exit

## Technologies
- Node.js + Express
- Socket.IO for signaling
- WebRTC for peer-to-peer video
- Vanilla JavaScript
- Mobile-first responsive CSS
